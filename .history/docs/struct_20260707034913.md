# ai-agent-core 项目结构

> Token-efficient Agentic Core — 确定性优先路由（cache → skills → MCP → LLM）+ 混合 RAG 检索 + 分级记忆

## 主要文件说明

### `agent.py` — 核心编排器

无状态的 `AgentCore` 类，是整个系统的入口和流程控制中心。

- **构造函数**接收 5 个路径（rules、routing、cache、short_term、long_term），初始化各模块
- **`handle(query)`** 是主入口，执行流程：
  1. 追加 `("user", query)` 到短期记忆
  2. 检查语义缓存（命中则直接返回）
  3. 路由匹配 → 调用对应 skill / mcp / llm
  4. 通过 evaluator 校验输出
  5. 失败时按 `fallback` 配置回退到 LLM
  6. 成功时写入缓存 + 短期记忆 + 长期三元组
- **`_parse_skill_args`** 把自然语言 query 解析成 skill 的 `args` 字典（支持 `calc`、`stats`、`read/load/show file`、`clean/sanitize file`）
- **`_call_llm`** 调用 Anthropic API，模型 ID 通过 `ANTHROPIC_MODEL` 环境变量配置

### `config/` — 配置层

- **`models.py`** — Pydantic v2 模型，开启 `extra="forbid"` 严格校验。`RulesConfig` 约束 `max_output_tokens ∈ [64, 8192]`，`output_format` 和 `tool_type` 使用 `Literal` 枚举
- **`loader.py`** — YAML 安全加载（`yaml.safe_load`），文件不存在抛 `FileNotFoundError`
- **`rules.yaml`** — 系统规则模板：角色为 "Senior AI Infrastructure Engineer"，Token 预算 1024，prompt 强制 JSON 输出
- **`routing.yaml`** — 路由表：`^(calc|compute|what is).*\d` → math_logic、`^(read|load|show|clean|sanitize).*file` → file_ops、`^(lookup|search|find).*` → knowledge MCP、`^(fetch|抓取|下载|crawl).*http` → fetch_web、`.*` → LLM 兜底

### `harness/` — 防幻觉层

- **`cache_guard.py`** — 语义缓存。Key = SHA256(lowercase + 折叠空白)，TTL 控制（`ttl_seconds=0` 表示立即过期），SQLite 存储。归一化保证 `"Hello World"` 和 `"  hello   world  "` 命中同一缓存
- **`evaluator.py`** — 输出校验器。检查信封 keys `{ok, result, error}` 齐全；`ok=False` 的失败信封直接透传；JSON 格式接受 dict/list/scalar/可解析 JSON 字符串

### `rag/` — 知识检索

- **`vector_db/store.py`** — 基于 sqlite-vec 的向量存储。`vec0` 虚拟表存 float 向量，搜索返回 `(id, text, 1.0 - distance)`（余弦相似度）。Upsert 用 DELETE+INSERT 绕过 vec0 主键限制
- **`retriever.py`** — 混合检索器。每次 query 时重建 BM25 索引（基于内存 docs 列表），同时调向量搜索，min-max 归一化后按权重 `0.5 * bm25 + 0.5 * vector` 融合，取 top-k
- **`corpus/`** — 知识库源文档目录，放 `.txt` / `.md` 文件

### `skills/` — 本地确定性技能

所有技能实现 `execute(args: dict) -> dict` 接口，返回 `{"ok": bool, "result": Any, "error": str | None}` 信封。

- **`base.py`** — `Skill` Protocol 定义 + `ok(result)` / `err(message)` 辅助函数
- **`file_ops.py`** — `read` 读取文件文本，`clean` 去除每行首尾空白并丢弃空行
- **`math_logic.py`** — `calc` 用 `ast.parse(mode="eval")` + 递归 `_safe_eval`（白名单 BinOp/UnaryOp/Constant），拒绝 `__import__('os')` 等 AST 节点；`stats` 返回 mean/sum/count
- **`fetch_web_to_md.py`** — 扩展技能：Web 抓取转 Markdown。**默认输出目录为 `rag/corpus/`**，抓取的内容直接进入知识库可被 `lookup` 检索；可通过 `output_path` 参数或环境变量覆盖

### `mcp/` — 协议化工具接入

- **`mcp_client.py`** — 工具注册表。`register(name, tool)` / `list_tools()` 返回排序后的名称列表 / `call(name, args)` 返回信封，捕获 tool 异常转为错误信封
- **`servers/knowledge_server.py`** — 知识库 MCP Server。加载 `corpus_dir` 下所有 `.txt`/`.md` 文件，`lookup` op 先做子串匹配，无匹配时用 BM25 兜底返回最相关文档

### `memories/` — 分级记忆

- **`short_term.py`** — 短期记忆。`deque(maxlen=N)` 滚动窗口，每次 append/clear 都持久化到 JSON 文件。支持跨实例加载（重启后状态保留）
- **`long_term.py`** — 长期记忆。SQLite 表 `triplets(subject, predicate, object, ts)`，支持按 subject/predicate 查询，`summarize_as_text()` 返回 `"subject predicate object"` 拼接文本
- 运行时的 `short_term.json`、`long_term.db`、`cache.db` 均被 gitignore

### `tests/` — 测试套件

65 个 pytest 测试，覆盖所有模块。AAA 模式（Arrange-Act-Assert），描述性命名（`test_<行为>`）。`test_e2e.py` 是端到端集成测试，覆盖 math skill / cache 命中 / file_ops / MCP lookup / 记忆状态 / LLM 兜底（无 API key 时返回错误信封）等完整流程。

### 顶层文件

- **`pyproject.toml`** — Python 3.11+，依赖 pydantic v2 / pyyaml / numpy / rank-bm25 / sqlite-vec / mcp / anthropic / python-dotenv。dev 依赖 pytest + pytest-cov + pytest-asyncio
- **`.env.example`** — 环境变量模板：`ANTHROPIC_API_KEY`、`CACHE_EMBEDDING_MODEL`、各模块路径
- **`.gitignore`** — 忽略 `.env`、`*.db`、`__pycache__/`、`.venv/`、`.pytest_cache/`、`memories/short_term.json`、`memories/long_term.db`
- **`README.md`** — 安装与使用说明（`python -m agent "What is 2+2?"`）

## 系统流程

```text
用户输入
  ↓
agent.py: handle(query)
  ↓
1. 追加到短期记忆
  ↓
2. cache_guard 检查 ─── 命中 → 直接返回
  ↓ 未命中
3. 路由匹配 routing.yaml
  ↓
4. ├─ skill  →  本地确定性执行（math / file_ops）
   ├─ mcp    →  外部工具调用（knowledge lookup）
   └─ llm    →  Anthropic API（fallback）
  ↓
5. evaluator 校验信封 + 格式
  ↓ 失败 + fallback=llm
   → 重试 LLM 路径
  ↓
6. 成功 → 写入 cache + 短期记忆 + 长期三元组
  ↓
返回 {"ok": ..., "result": ..., "error": ...}
```
