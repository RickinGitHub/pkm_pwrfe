# 60、新方案合规评估

> pageId: 243464247 | 导出时间: 2026-07-07T14:52:41.548477

角色A更新时间
 
沟通对象安全合规测试人员，安全合规经理，软件开发工程师、软件开发经理具体内容方案合规评估操作方法
1、需注意方案用到的文件夹权限是否合规。

2、需注意方案用到的数据分区是否合规。

3、应用开发，需检查依赖的第三方库是否涉及隐私安全，官方库是否披露安全隐患。

4、应用开发，涉及获取权限，注意满足GDPR要求。

5、如果是第三方应用，需注意是否签了免责合同，并且上线谷歌商店了。
