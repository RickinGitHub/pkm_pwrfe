# 2. 必解问题梳理

> pageId: 247458548 | 导出时间: 2026-07-07T14:53:19.051444

角色A更新时间
 
沟通对象质量SE、VPM、SPM具体内容梳理必解问题（IP前2个月开始）操作方法
1、遍历jira上的问题，熟悉各问题的基本情况及进展；

2、通过SPM组织质量SE、VPM开展必解问题梳理会议；

3、会议评审过程中，按以下标准梳理必解问题：

    a、系统或各应用模块出现无法启动或异常退出或功能完全失效（如无法搜台、CA无效、Launcher不显示、VOD无法播放、关键应用无法运行）

    b、严重影响用户使用的问题，如常用路径下，系统或模块崩溃、卡死、黑屏、花屏、重启、死机、持续性卡顿、不响应遥控器等用户不可操作，或用户不可终止的错误状态（如持续弹窗）；

    c、严重的画质问题（图像水印，图像花屏，图像严重拖尾，图像闪等影响用户使用）；

    d、用户数据丢失、失效/扣费异常/会员失效，资金/权益受损；

    e、重要数据、参数和配置信息损坏，无法恢复导致用户数据记录错误/丢失；

    f、强制性认证问题（认证Owner 梳理的认证必解问题）

    g、强制性/法规要求内的软件安全合规性，数据隐私相关问题

    h、软件使用过程中造成硬件损坏或硬件问题需要软件改善的

    i、涉及国家政治、法规、文化、宗教政策等内容，以及暴力、色情等违法内容

    j、品牌相关信息错误

    k、影响卖场销售演示

    l、影响工厂生产的生产适应性问题

4、给必解问题打上必解标签，由SPM制定解决计划，推进必解问题迅速收敛，并在IP前收敛所有必解问题；
标准与原则影响项目释放的问题必解适用项目阶段LR、TR、PR适用项目范围S、A、B交付件必解问题表备注
