# 50、返工及客户软件管理

> pageId: 243464233 | 导出时间: 2026-07-07T14:52:26.933443

角色A更新时间
 
沟通对象产品开发经理、测试经理、产品所领导、测试leader、QPL具体内容  释放返工软件或者客户软件给到工厂生产操作方法
1、对需要返工的问题，上升产品所领导、产品开发经理、测试经理、QPL决策，确定需要返工后，协调开发资源对返工问题或客户反馈的问题解决后，新软件需要SQA测试、工厂FQA测试或者客户确认后，按软件释放流程完成签字流程

2、测试leader将软件上传到对应的工厂FTP目录
3、软件leader给出返工软件路径，给客户的软件需要给出下载链接及使用说明
标准与原则提供返工软件路径或者客户需要的下载链接及使用说明适用项目阶段
kick off
适用项目范围S，A，B，C，D交付件
