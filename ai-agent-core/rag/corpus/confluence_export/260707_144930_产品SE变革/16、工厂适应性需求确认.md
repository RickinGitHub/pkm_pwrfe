# 16、工厂适应性需求确认

> pageId: 243464184 | 导出时间: 2026-07-07T14:51:42.378953

角色A更新时间
 
沟通对象AOE工程师、软件开发经理、硬件工程师，合作团队软件接口人具体内容工厂生产适应性的具体需求确认和方案输出、相关的工厂串口和遥控协议（相关的串口、工厂遥控协议文档个、工具上传到confluence上，在此处增加链接）操作方法1、根据项目的需求规格书，与模块Owner（包含工厂模块）、AOE工程师梳理生产适应性相关的需求（要注意新增功能、外设等生产测试的需求），确认对应的调试流程、串口协议和工厂遥控协议；
2、若生产适应性的需求涉及生产效率的改善或新增功能的实现，则需组织工厂模块Owner、系统工程师等参与评估，并制定输出技术方案和计划；
3、若生产适应性的需求涉及外设的测试，则需硬件、声学、PQ等工程师评估相关需求； 
4、对于多方合作开发项目（例如Roku/Titan），需与AOE工程师一起确认由合作方提出的工厂适应性需求，并讨论对应的解决方案。标准与原则
