| **文件** | `./[精华]亲密关系-亲密关系中的情绪波动与冲突管理.md` |
| **▸ 女性生理周期·生化机制** | ① **雌激素→血清素"断崖式离场"**：黄体期后期（月经前7-10天）雌激素骤降→血清素合成减少→情绪调节能力暂时性下降——**这不是"作"，是生理** ② **PMS**（经前综合征）：70-90%女性有不同程度症状——易怒/焦虑/疲劳/失眠 ③ **PMDD**（经前焦虑障碍）：DSM-5正式收录·5-8%女性符合诊断标准——情绪影响已达**临床级别** |
| **▸ 男性IMS·对应机制** | ① **睾酮日周期**：晨高夜低——早晨睾酮最高时情绪最稳定 ② **睾酮年周期**：秋季最高·春季最低 ③ **压力效应**：皮质醇↑→抑制睾酮分泌→易怒/疲惫/低欲望——**这是男性版的"生理期"，只是周期更长更隐蔽** |
| **▸ 三大心理学原理·实战应用** | ① **认知评价理论**：生理不适→外部归因（"都是因为你"）→情绪爆发。**对策**：当对方情绪爆发时，先想"TA是不是饿了/累了/在生理期？"——做**生理诊断**而非**人格攻击** ② **自我耗竭**（Ego Depletion）：白天消耗自控力（工作/社交/决策）→晚上对伴侣失控。**对策**：晚上9点后不做重大关系讨论——自控力已耗尽 ③ **躯体化**：心理痛苦转化为身体症状——头疼/疲劳/失眠，但根源是情绪。**对策**：当对方反复抱怨身体不适但查不出原因→可能是情绪问题在躯体化 |
| **▸ 实战三原则** | ① 识别生理周期节奏→在"高危窗口"（经前7-10天）降低冲突期望·不讨论敏感话题 ② 区分"激素驱动"和"真实矛盾"——生理期的问题**生理期后再说**（如果生理期后问题消失了=激素驱动·如果还在=真实矛盾）③ 建立"情绪缓冲区"——感觉要爆发时先**物理隔离15分钟**（去另一个房间/出门散步） |
